# Task Management System

## Installation

1. Clone this repository to your local machine.

2. Install the required PHP packages using Composer:

   ```bash
   composer install

3. Install and Build the JavaScript packages using npm: npm install && npm run build

4. Copy the .env.example file to .env: cp .env.example .env
 
5. Generate an application key: php artisan key:generate

6. Set up your database connection by editing the .env file and filling in the necessary database credentials.

7. un the database migrations and seed the database: php artisan migrate --seed

## Usage

To access the admin panel with tasks, you can log in using the following default credentials:
Email: admin@admin.com
Password: 'admin'

Alternatively, you can register a new user account using the /register route.
By default, tasks are associated with each user individually.
